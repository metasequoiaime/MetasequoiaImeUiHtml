开始使用水杉 IME

水杉 IME 是一款面向 Windows 的中文输入法。你可以在这里了解基础输入流程、候选操作以及各项扩展功能。

快速上手

安装并启用输入法后，在中文状态下直接输入拼音。使用数字键或空格选择候选，按 Esc 取消当前输入。

输入方案

支持全拼、双拼和五笔。双拼方案与辅助码可在对应设置页调整，切换方案后会自动重建输入会话。

候选与翻页

支持横排、竖排候选框，以及多种翻页和方向键导航方式。相关按键可在“输入”和“快捷键”中配置。

扩展功能

语音输入、屏幕键盘、快捷短语、云候选和 AI 联想均为独立功能，可按需启用并配置服务接口。

常见问题
候选窗口没有出现确认当前处于中文状态，并检查输入方案及候选窗口相关设置。
设置没有立即生效部分底层设置需要重新建立输入会话；必要时可重启输入法服务。
联网功能响应较慢检查网络、接口地址和 Token，并确认所选模型支持当前 API 格式。
