# Metasequoia IME UI HTML

This is html&css&js codes of [MetasequoiaIME](https://github.com/fanlumaster/MetasequoiaIME).

For debug:

```shell
git clone https://github.com/fanlumaster/MetasequoiaImeUiHtml.git
New-Item -ItemType SymbolicLink -Path "C:\Users\<username>\AppData\Local\MetasequoiaImeTsf\html" -Target ".\MetasequoiaImeUiHtml"
```

e.g.

```shell
git clone https://github.com/fanlumaster/MetasequoiaImeUiHtml.git
```
