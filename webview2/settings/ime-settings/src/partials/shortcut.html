快捷键
输入法快捷键仅在对应输入状态或候选窗口显示时生效。翻页方式可在“通用”中启用或关闭。
中英文切换
Shift 由水杉输入法处理；Ctrl+Space 由 Windows 管理
水杉输入法内切换中 / 英文
Shift
修改或关闭 Ctrl+Space
请前往 Windows 系统设置修改“输入法 / 非输入法切换”的按键顺序：
  1. 打开“设置”，进入“时间和语言” → “输入”。
  2. 选择“高级键盘设置” → “输入语言热键”。
  3. 选中“中文（简体）输入法 - 输入法 / 非输入法切换”，点击“更改按键顺序”。
  4. 关闭该按键顺序，或将 Ctrl+Space 改为其他不常用组合。
不同 Windows 版本的选项名称可能略有差异；修改后如未立即生效，请重新登录或重启电脑。
